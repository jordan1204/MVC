專業資訊服務

  • 企業應用系統開發

  • 網星資訊成功承接各大企業的應用系統開發專案，抱括MES、CRM、WMS等等系統。我們的專案團隊嚴謹遵守 Software Development Methodology，除了功能亦強調效能與品質，嚴謹的客戶資料保密流程是網星資訊所堅持的，對於客戶所提供的資料，我們有一套完整的控管制度，以保障客戶資料的機密性。

  • 效能調校服務

  • 透過應用程式效能管理產品能持續監控效能問題，讓應用程式和資料庫執行速度更快，一旦發現系統變慢，就能迅速進行診斷並修正問題。
  • Performance tuning for database (Oracle, SQL etc.)
  • Performance tuning for application (J2EE, .Net etc.)

  • 專業資訊顧問服務

  • 網星資訊的專案經驗包括︰IT服務管理的建置與營運、IT Infrastructure建置、工廠製造執行系統(MES)等等。

  • 行動應用軟體App開發服務

    •
  • 網星資訊擁有行動App應用軟體與雲端服務兩類產品的設計與開發經驗，可交互應用兩類產品的研發經驗，從App概念的發想、美術設計、程式開發等，皆有專業的人才提供最適切的服務。
